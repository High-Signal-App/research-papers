# Current

Live operating state lives in [`STATUS.md`](../../STATUS.md) at the repo root
(short view) and in [`PROJECT_STATUS.md`](../../PROJECT_STATUS.md) (fleet-facing
product identity + timeline). This folder holds current-cycle artifacts that
are not yet ready to be archived.

## Pages

- [`timeline.md`](timeline.md) — chronological log of shipped changes, mirrored
  from `PROJECT_STATUS.md` → Timeline. The canonical source is
  `PROJECT_STATUS.md`; this page exists so Blume surfaces it in the docs site.

## What does NOT live here

- Durable product identity → [`../product/`](../product/index.md)
- Architecture decisions → [`../architecture/decisions/`](../architecture/decisions/index.md)
- Learnings → [`../knowledge/learnings.md`](../knowledge/learnings.md)
- Superseded material → [`../archive/`](../archive/index.md)
